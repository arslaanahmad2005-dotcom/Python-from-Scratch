#input() function

The input() function captures user data from the keyboard and always returns it as a string (str).

#Type Casting

Type casting explicitly converts a variable from one data type to another. It is necessary when processing mathematical inputs because input() defaults to text.

#Concatenation

Concatenation is the process of joining two or more strings together using the + operator. It requires manual spacing and fails if you try to join a string with a number without type casting.

#F-Strings Method

Formatted string literals (f-strings) embed variables or expressions directly inside a string using curly braces {}. It is the cleanest, fastest, and most recommended method in modern Python.
Prefix the string with an f or F
