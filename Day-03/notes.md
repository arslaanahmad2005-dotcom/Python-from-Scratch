#Comparison Operators

Comparison operators are used to compare two values. They always return a Boolean value: True or False. These operators help a program make decisions.

== : Equal to
!= : Not equal to
> : Greater than
< : Less than
>= : Greater than or equal to
<= : Less than or equal to

#if statement

The if statement is used to execute a block of code only when a specified condition is True. Python uses indentation (4 spaces) to identify the code inside the if block.
#Syntax:
if condition:
    # Code to execute if condition is True

#Handling Alternatives (elif and else)

When there are multiple conditions, Python uses elif and else.
elif (else if): Checks another condition if the previous if or elif condition is False.
else: Executes when none of the previous conditions are True.
