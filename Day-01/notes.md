#Variable: A temporary storage location or container where we can store any value.

#Datatypes: A type of data which we store is called its datatype.
#Types of datatypes: int, float, String, boolean etc.

#Dynamic Typing: It means you do not need to declare a variable's datatype explicitly because Python automatically determines is at runtime based on the values assigned.

#type(): this is a function used to check the datatype  of any variable.
